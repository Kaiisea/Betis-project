Programando desde los bordes

Un proyecto de programación alternativa, donde lo que se comunica se hace desde la disidencia de lo normativo. Soy una persona experimental, me encanta dar más de lo que me piden, hacerlo todo desde un punto de partida diferente.
Soy una desarrolladora web Full Stack que por ahora pretende tocar todos los palos posibles. Estoy empezando con lo básico del Front-End: HTML, SCSS y Javascript. Uso compiladores y frameworks como SASS, Bootstrap y Javascript DOM, usados para hacer este propio portfolio. Para desarrollar este proyecto tambien me he apoyado en Git Hub para el control de versiones y Visual Studio Code para editar el código.
